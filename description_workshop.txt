[h1]Oink![/h1] 

[h1]This is a mod that allows users to transform the Calamity Mod's Piggy NPC into a little companion.[/h1] 

[h2]Should you not have calamity installed, some recipes for the pet are included using vanilla items.[/h2]

[hr][/hr]
[h3]Features include:[/h3]

[olist]
    [*]Piggy
    [*]Jumping for joy with Piggy
    [*]Giving Piggy a fruit
    [*]IRL happy buff (no bonus to speed included)
    [*]Slaying Gods with Piggy
    [*][spoiler]Oink[/spoiler]
[/olist]

This mod's source code can be found [url=https://github.com/BrinkOfMan/PetPiggy]here[/url]
